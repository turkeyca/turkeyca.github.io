C++数据结构之栈

栈（stack）又名堆栈，它是一种运算受限的线性表。其限制是仅允许在表的一端进行插入和删除运算。这一端被称为栈顶，相对地，把另一端称为栈底。
类似线性表，我借用了模板类来方便处理各种类型的数据

5.26更新：
C++中STL包含了栈这一结构，头文件为#include,创建栈为stackistack,入栈为istack.push,c出栈为istack.pop,获取栈顶元素为istack.top,判断栈是否为空 istack.empty(bool).
